# Настройка переменных окружения

## Файлы .env

### `.env` - для продакшена (Netlify)
```
VITE_API_BASE_URL=https://85858ce02571.ngrok-free.app
```

### `.env.local` - для локальной разработки (приоритет над .env)
```
VITE_API_BASE_URL=http://localhost:8000
```

## Как это работает

Vite загружает переменные в следующем порядке (приоритет):
1. `.env.local` - локальная разработка (игнорируется git)
2. `.env` - общие настройки

## Для локальной разработки:

1. Убедитесь, что `.env.local` существует с `VITE_API_BASE_URL=http://localhost:8000`
2. Убедитесь, что бэкенд запущен: `python3 manage.py runserver 0.0.0.0:8000`
3. Перезапустите dev-сервер фронтенда после изменения .env файлов

## Для продакшена (Netlify):

1. Используйте `.env` с ngrok URL
2. Или настройте переменную окружения в Netlify: `VITE_API_BASE_URL`
